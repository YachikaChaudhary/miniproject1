# miniproject1

EXPLANATION - 
This project is a simple e-commerce application. On the home page, we fetch products from the DummyJSON API and dynamically create product cards using JavaScript. Clicking a product sends its ID through the URL to the product page, where we fetch and display its details. When the user clicks 'Add to Cart', the product is stored in Local Storage. The cart page reads the stored data, displays all selected products, calculates the total number of items and total price, and also allows users to remove items from the cart. CSS is used for layout and styling, while Local Storage is used to persist cart data between pages.
----------------------------------------------------------------------------------------------------------------------------------PROJECT FLOW -
  This is a basic E-commerce Cart Application built using HTML, CSS, JavaScript, Fetch API, and Local Storage.

  The project has 3 pages:
1. index.html – Displays all products.
2. product.html – Shows details of one selected product.
3. cart.html – Displays products added to the cart and the bill summary.
